# Logo Assets

Please place your HealthGuard logo and icon files here:

## Required Files:

1. **app_icon.png** - Main app icon (recommended: 1024x1024px)
   - This will be used for the splash screen and app icon
   - Should be a square image with the HealthGuard shield logo

## File Structure:
```
assets/
  logo/
    app_icon.png  (Place your logo here)
  images/
    (Other image assets)
```

## Notes:
- The splash screen will automatically use `app_icon.png`
- If the file is not found, it will fall back to a default health icon
- Recommended format: PNG with transparent background
- Recommended size: 1024x1024px for best quality
